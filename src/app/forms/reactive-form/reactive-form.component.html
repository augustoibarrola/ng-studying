Reactive Forms
Reactive Forms (also called Model Drive Forms) are best suited for Large forms (i.e., forms with more than two or three simple input fields and labels).

Reactive Forms work by creating form control objects in the component class of a template and binding those controls with HTML form elements in the template.

FormBuilder from `@angular/forms` is used to create Reactive Forms within the component class.

FormGroup, also from `@angular/forms`, is used to name and represent the form and its values inside the component class by treating it as a FormGroup instance.

ReactiveFormsModule from `@angular/forms`must be imported in the corresponding module of the form's component and then placed inside the imports:[] field of @NgModule in order to use Reactive Forms in Angular. ReactiveFormsModule replaces FormsModule when creating reactive forms.

[formGroup]="registerForm" is a directive that binds the HTML form tag with the FormGroup property created in the component class (i.e., registerForm: FormGroup). This connects the form and all it's children values to the property in the template's corresponding component class.

When a submit button is clicked in a reactive form, the form is submitted through its ngSubmit directive, initializing the form's submitted proeprty to true. Hence, (ngSubmit)="onSubmit()", indicating on that, upon submission, the onSubmit() function within the component class will be executed.
