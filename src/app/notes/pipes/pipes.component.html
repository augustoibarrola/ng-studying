Pipes

Multiple pipes can be used at a time on a single expression, as long as the pipes are compatible with each other.


Built-In Angular Pipes

  • expression | uppercase
  • expression | lowercase
  • expression | titlecase
  • expression | currency
  • expression | date
  • expression | percent
  • expression | slice
  • expression | decimal
  • expression | json
  • expression | i18nplural
  • expression | i18nselect

Custom Pipes

transform() method has two arguments, first one is the value of the expression passed to the pipe and the second is a variable "arguments". We can have multiple arguments based on the number of parameters passed to the pipe. The transform method should return the final value.

Custom made pipes need to be added to the declarations:[] property of the file's respective @NgModule file.
